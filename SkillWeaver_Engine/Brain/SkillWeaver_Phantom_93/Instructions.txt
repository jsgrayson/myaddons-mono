
SKILLWEAVER v22.0 (STEALTH PHANTOM) - COMMAND CENTER

== SETUP INSTRUCTIONS ==

1. KEYBINDS (MUST MATCH IN-GAME):
   - F12: Interupt (Target)
   - F10: Arena 1 Kick
   - F11: Arena 2 Kick
   - F09: Arena 3 Kick
   - F08: Focus Kick
   - F19: /cast [@arenapet1] Taunt  (For Scatter Shot Break)
   - F18: /use 13 (Trinket 1)
   - F17: /use Healthstone
   - F20: /cast Call Pet 1
   
2. GROUND TARGET MACROS (REQUIRED):

#showtooltip Shadowfury
/cast [@cursor] Shadowfury

   
3. ADDON INSTALLATION:
   - Copy SkillWeaver_Addon.lua to Interface/AddOns/SkillWeaver/
   - Create a .toc file.
   - Reload UI.

4. LOGIC ENGINE:
   - Install Python 3.9+
   - pip install mss numpy pyserial pillow
   - python LogicEngine.py

== STEALTH PROTOCOL ==
   This version uses Differential RGB encoding. 
   The bar at the bottom will look like a solid dark gray line.
   Do not change Gamma/Brightness settings while running.
